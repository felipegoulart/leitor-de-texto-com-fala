# Aplicação de leitor de texto com fala

## Para quê?
A aplicação permite que pessoas com dificuldade de fala possam se comunicar com mais facilidade.

## Como usar?
Acesse https://leitor-de-texto.vercel.app/ e clique nas imagens/texto para executar a fala.
Você pode também digitar um texto clicando em "Insirir Texto" e depois clicar em "Ler Texto"

Este projeto foi realizado com o tutorial do Roger Melo https://www.youtube.com/watch?v=OY21UokkEYk&list=PLpSJMw6H4PFMOJHMULTxKNOEw7g1cBuyP&ab_channel=RogerMelo.
